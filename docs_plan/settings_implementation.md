# 설정 기능 구현 및 시스템 연동 계획

설정 페이지의 가짜 UI들을 제거하고, 사용자의 선택이 사이트 전체에 반영되도록 실제 로직을 연동합니다.

## 1. 전역 설정 엔진 (`settings-engine.js`)
* 로컬 스토리지 데이터 관리 (Load/Save)
* 다크모드/라이트모드 테마 전환 엔진 구현
* 배경 애니메이션 (Aurora 효과) 제어 로직
* 효과음 전역 상태 관리

## 2. 설정 페이지 개편 (`settings.html`)
* `disabled` 처리된 체크박스 활성화
* `saveSettings()` 함수 고도화 (데이터 저장 및 즉각 반영)
* 설정값 로드 및 UI 초기화 로직 추가

## 3. 메인 및 개별 도구 페이지 연동 (`index.html` 및 CSS)
* 페이지 로드 시 설정 엔진 호출
* 배경 애니메이션 제어 클래스 적용
* 테마 변수 업데이트
